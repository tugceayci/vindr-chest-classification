# VinDr Chest Classification

Deep learning based chest X-ray anomaly detection project using the VinDr-Chest dataset.

## Objective
The goal of this project is to classify chest X-ray images as:
- Normal
- Abnormal (Finding)

using deep learning techniques and transfer learning approaches.

## Dataset
VinDr-Chest X-ray Dataset
